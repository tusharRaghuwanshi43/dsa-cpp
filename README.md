# dsa-cpp 🧠

> A personal C++ repository to revisit, revise, and strengthen DSA concepts and programs.

## 🎯 Purpose

This repository serves as a personal revision kit for Data Structures and Algorithms.
The goal is to write, revisit, and reinforce core DSA concepts through consistent
practice in C++.

## 🛠️ How to Run

Make sure you have a C++ compiler installed (e.g., g++).

```bash
# Compile
g++ filename.cpp -o output

# Run
./output
```

## 📁 Structure

All programs are written in C++ and follow a simple naming convention:
`topic-subtopic.cpp` — making it easy to find and revisit any concept.

## 🔄 Consistency

This repo is actively maintained and updated regularly as new
concepts and problems are practiced.
